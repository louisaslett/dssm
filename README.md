# MATH2801: Data Science and Statistical Modelling

This repository provides a [Github Codespaces](https://github.com/features/codespaces/) for running the R computer labs in the [MATH2801 module](https://www.dur.ac.uk/faculty.handbook/module_description/?year=2025&module_code=MATH2801) in the Department of Mathematical Sciences at Durham University, delivered by [Prof. Aslett](https://www.louisaslett.com/) and [Prof. Coolen-Maturi](https://tahanimaturi.com/).
